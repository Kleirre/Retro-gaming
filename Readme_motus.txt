Ce programme a besoin du fichier 'dictionnaire_motus.txt' pour fonctionner. Merci de ne pas le retirer du dossier.

Fonctionnalités du programme :

Ce programme reproduit le fonctionnement du jeu télévisé Motus. L'utilisateur choisit le nombre de lettres du mot qu'il veut deviner, puis le nombre maximum de tentatives auquel il aura droit. Le programme affiche ensuite la première lettre du mot suivie de tirets. L'utilisateur enntre alors un mot qu'il pense être la bonne réponse, si c'est le cas, il a gagné, sinon les lettres bien placées s'afficheront en majuscule et les lettres mal placées s'afficheront en minuscule, si elles n'appartiennent pas au mot, elles seront remplacées par de simples tirets.
À chaque fin de partie, l'utilisteur a la possibilité de rejouer.
Tous les mots proposés ne contiennent pas d'accents ou autres caractères spéciaux.

Pour lancer le programme, exécutez-le dans le shell.

Programme réalisé avec Pyzo 3.12.3
et Python 3.12.3
