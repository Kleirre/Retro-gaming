Ce programme a besoin des trois fichiers : demineur.py, initialisation.py, jeu.py et scores.txt pour fonctionner. Merci de ne pas les retirer du dossier.

Fonctionnalités du programme :
Ce programme est un jeu du démineur. Le joueur peut tenter de trouver des mines disposées aléatoirement dans une grille. Il peut révéler des cases mais aussi poser des drapeaux pour identifier ces mines.

Pour lancer le programme, exécutez le fichier demineur.py dans le shell.

Programme réalisé avec Pyzo 3.12.3
et Python 3.12.3

Ce programme passe les tests pylint.
Ayant changé de moyen-proet sur le tard, nous avons eu peu de temps pour achever ce démineur. Nous avons fait dans le temps imparti ce que nous avons pu. Il se peut qu'il subsiste des bugs.

Bonus:
- fonction creuser intelligente (appel récursif)
- l'utilisateur peut poser des drapeaux (et pas uniquement creuser)
- l'utilisateur peut personnaliser le nombre de mines
- l'utilisateur peut personnaliser le nombre de cases
- l'utilisateur peut jouer autant de parties qu'il veut sans avoir besoin d'exécuter à nouveau le code
- l'utilisateur peut consulter les règles du jeu (si il le souhaite)
- les scores sont inscrits dans un fichier et peuvent être affichés par l'utilisateur
- ajout d'une licence
